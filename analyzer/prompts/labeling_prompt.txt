# Output Format

# Critical Logic Protocols (核心逻辑协议)

## 1. Validity Filter (有效性过滤逻辑)
在开始分析前，先判断文本是否具备分析价值：
- **noise**: 纯表情包堆叠、乱码、纯外部链接、无情感倾向的重复字符（如 "11111"）、系统自动转发。 -> **直接输出 validity="无意义-噪音"，其余字段均为 null**。
- **stanning_and_other_spamming**: 即使内容短、重复、堆砌关键词，只要其目的是为了**增加偶像数据权重**、**表达支持**或**占领话题广场**（如“抱走不约”、“期待哥哥新剧”、“专注自家”），均属于**有效内容**，必须进行完整分析！
- **general**: 具有一般语义的观点表达、叙事。

## 2. Contextual Decoupling (话题-正文解耦逻辑)
微博内容结构为：`话题标签 (##)` + `正文内容`。
- **判读原则**: 必须基于**正文内容**来判断博主的立场和情绪。
- **蹭热度判定**: 如果正文内容是记录生活（如“今天吃火锅”），但携带了负面或宏大话题（如 #某明星出轨# #国际局势#），且正文与话题**毫无语义关联**，则 **Stance 必须标记为 "无关"**，**Target 标记为 "其他"**。
- **话题响应**: 如果正文是对话题内容的直接评价（无论支持还是反对），则视为有关。

## 3. Implicit Causality & Attribution (隐性因果与归因逻辑) **(重要)**
博主往往不会直说“因为A所以B”，而是通过并列描述暗示因果。
- **潜规则/不正当竞争**: 如果博文暗示某人获得利益（如获奖、上位）是因为“陪睡”、“资本”、“后台”而非“实力”，即使使用了侮辱性词汇（如“野鸡”、“整容怪”），其核心道德逻辑依然是 **Fairness/Cheating (破坏了公平竞争)**，而非单纯的 Sanctity。
- **优先级**: 当 Fairness (不公) 与 Sanctity (恶心) 同时存在时，优先判定为 **Fairness/Cheating**，因为社会不公的权重更高。

## 4. Safety Valve (安全阀) **（重要）**
- **拒接强行推理**: 若文本指代不清(如"那件事真无语"但未提何事)、或情感/道德指向极其模糊，请直接选 **Uncertain**。不要过度解读。

# Annotation Codebook (标注代码本)

请从以下列表中选择最精准的标签：

## 1. Stance (博主立场)
- **support**: 明确赞同、喜爱、声援、安利、辩护。
- **oppose**: 明确抵制、批判、厌恶、嘲讽。
- **neutral**: 客观陈述事实，不带感情色彩的“吃瓜”。
- **no_clear_stance**: 纯粹的情绪宣泄（如“今天好烦啊”）、生活记录，未针对特定对象表态。
- **irrelevant**: 正文与话题标签完全无关（蹭热度）；或携带多个矛盾话题导致指向不明。
- **uncertian**: 信息严重缺失，无法推断意图。

## 2. Emotion_Category (情感大类)
[anger, disgust, annoyance, like, sadness, happiness, fear, surprise, none, uncertain]

## 3. Emotion_Subtype (情感子类 - 请细化)
- **anger**:
    - **personal_anger**: 个人生活中的生气。
    - **moral_outrage**: 对违反道德、社会不公的愤怒（MFT分析的重点）。
    - **fury**: 攻击性极强的情绪。
- **disgust**:
    - **physical_disgust**: 脏、丑、想吐。
    - **moral_contempt**: 看不起人品、下头、三观不正、阴阳怪气。
    - **rejection**: “抱走不约”、“别来沾边”等饭圈现象，以及其他较为明显的拒绝别人的请求、误解、邀请、或者和某一群体割席。
- **like**:
    - **idol_doration**: 饭圈专用，神化偶像，“杀我”、“神颜”、“yyds”。
    - **simple_liking**: 觉得不错。
    - **admiration**: 对专业能力或品德的尊敬。
    - **solidarity**: 感动、支持。
- **annoyance**: [annoyance]
- **sadness**: [disappointed, empathy, grieve, hopeless]
- **happiness**: [simple_happy, pride, gloat, looking_forward_and_hope]
- **fear**: [anxious, fear, concerned]
- **surprise**: [astounded, happily_surprised, shock]
- **none**: [general]

## 4. Target (情感对象)
[person, outfit_or_appearance, action, opinion, group_or_organization, works_or_creations, scenery, others]

## 5. MF_Main (道德基础大类)
[care_vs_harm, fairness_vs_cheating, loyalty_vs_betrayal, authority_vs_subversion, sanctity_vs_degradation, none, uncertain]
*判例指南*:
- **fairness_vs_cheating**: 涉及权利、正义、**作弊**、**潜规则**、特权、双标。
- **sanctity_vs_degradation**: 涉及身体/精神纯洁、**性伦理**、恶心、传染病、亵渎。
- **loyalty_vs_betrayal**: 涉及国家、团队、家庭的背叛（如汉奸、私生饭背叛偶像隐私）。
- **authority_vs_subversion**: 涉及遵守规则、敬业、尊重传统/长辈。
- **care_vs_harm**: 涉及同情、暴力、保护弱者。
- **none**: 纯审美评价（如“好帅”）、无道德评价。

## 6. MF_Direction (道德具体方向)
[care, harm, fairness, cheating, loyalty, betrayal, authority, subversion, sanctity, degradation, none, uncertain]

## 7. Reasoning (推理链)
请用一句话简述判定理由。**必须指出因果关系**（例如：“因为[事实A]，导致了[后果B]，触犯了[MF]”）。
